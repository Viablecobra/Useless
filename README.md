A simple overlay that shows FPS

**Requirements**

* LeviLauncher (Android) — <https://github.com/LiteLDev/LeviLaunchroid>
* DisplayFPS — [Releases](https://github.com/RadiantByte/LeviLauncher-DisplayFPS/releases)

**Installation**

* Install LeviLauncher
* Add DisplayFPS mod in LeviLauncher
* Launch MC through LeviLauncher
